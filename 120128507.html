Metodika pre otvorené údaje (opendata.gov.sk) : Load NKODu do vlastného triplestore

Created by miroslav.liska, last modified on nov 04, 2021

Keďže súčasný data.gov.sk neposkytuje funkčný SPARQL Endpoint, je nutné naištalovať si vlastnú RDF databázu, do nej nahrať NKOD, na základe čoho je ho možné dopytovať v jazyku SPARQ.

Postup je nasleodný:

  1. Stiahnite si RDF4J Server + Workbench , naištalujte, nech beží ako webový server
  2. Vytvorte si novú databázu data.gov.sk

3. Naimportuje si NKOD, ktorý získate tu: Prístup k NKOD

4.  Odteraz môžete databázu s NKODom dopytovať. Použite Príklady SPARQL dopytov pre NKOD
